# ProfileListItem

Renders a list item for a profile overview.

```tsx
import { ProfileListItem } from "@lens-protocol/react-native-lens-ui-kit";
<ProfileListItem profile={profile} />;
```

## Default props

```
profile (required)
isFollowing
onProfilePress
onFollowPress
```
